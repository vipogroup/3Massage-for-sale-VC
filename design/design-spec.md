# VIPO — אפיון עיצוב הדמו

> דף מוצר לכורסת עיסוי **VC LUXURY**.
> מסמך זה מתעד את מערכת העיצוב של הדמו כפי שמומשה ב-`hero-lab.html` + `css/hero-lab.css`.
> **סטטוס: דמו בלבד** — sandbox בתיקיית `design/`. אין העברה ל-`index.html` ללא אישור מפורש.

---

## 1. קונספט עיצובי

| עיקרון | יישום |
|--------|--------|
| **Mobile-first** | רוחב מקסימלי `480px`, ממורכז. כל המידות מותאמות למסך נייד. |
| **Premium / יוקרתי** | פלטת navy + gold, glassmorphism, צללים רכים, מעברי gradient. |
| **Trust-driven** | badges אמון, אחריות, תקן CE, ביקורות מאומתות — לאורך כל הדף. |
| **Urgency** | טיימר LIVE, מלאי שנותר, פס התקדמות, "המכולה בדרך". |
| **Unified flow** | רקע gradient זורם, מחברים בין סקשנים, scroll-reveal — חוויה אחת רציפה. |

## 2. Design Tokens

### 2.1 צבעים (CSS Variables)

```css
--navy:     #1c2b46   /* צבע מותג ראשי, טקסט כותרות, CTA */
--navy-2:   #243a5e   /* navy בהיר יותר — gradients */
--gold:     #b08d57   /* אקסנט יוקרה, badges, קישוטים */
--green:    #1a6b4a   /* badge "רכישה מאומתת" */
--green-bg: #e6f4ee   /* רקע ירוק רך */
--ink:      #1c2b46   /* טקסט ראשי (= navy) */
--ink-2:    #5b6472   /* טקסט משני */
--ink-3:    #9aa3b2   /* טקסט מעומעם / placeholder */
--line:     #e6e9ee   /* קווי הפרדה, גבולות */
--bg:       #ffffff   /* רקע ראשי */
--bg-alt:   #f8f9fb   /* רקע משני (סקשנים מתחלפים) */
```

**צבעי אייקונים סמנטיים (סקשן יתרונות):**
- בריאות: רקע `#fef0f0`, אייקון `#d63031` (אדום)
- רוגע: רקע `#edf7f0`, אייקון `var(--green)`
- אחריות: רקע `#eef1f8`, אייקון `var(--navy)`
- מפעל: רקע `#fdf5eb`, אייקון `var(--gold)`

**צבעי דירוג כוכבים:** `#f0b429` (זהב-צהוב).

### 2.2 טיפוגרפיה

- **גופן:** `Assistant` (Google Fonts) — משקלים 400/500/600/700/800. fallback: system sans.
- **כיוון:** RTL (`dir="rtl"`, `lang="he"`).
- **סולם גדלים נפוץ** (rem): מחיר ראשי `1.6`, כותרת hero `1.15`, כותרות סקשן `0.98`, גוף `0.66–0.72`, micro `0.5–0.6`.
- **משקלים:** כותרות `800–900`, תוויות/badges `700–800`, גוף `500–600`.
- **תווים מספריים:** `font-variant-numeric: tabular-nums` בטיימר ובסטטיסטיקות (יישור יציב).

### 2.3 צורה ומרחב

- **Border radius:** טוקן `--r: 14px`. כרטיסים גדולים `16–18px`, footer עליון `24px`, אלמנטים קטנים `8–12px`, עגול מלא `999px`.
- **Padding סקשן סטנדרטי:** `20px 14px`.
- **Gap בין כרטיסים:** `8–12px`.
- **Spacing bottom של body:** `80px` (פינוי לכפתור הצף).

## 3. מבנה הדף (Sections)

הדף בנוי כ-`header` דביק + `main` (10 סקשנים) + `footer` + כפתור צף דביק.

| # | מזהה | תוכן | אלמנט מפתח |
|---|------|------|------------|
| 0 | `.vp-header` | הדר דביק | לוגו ממורכז, וואטסאפ, שיתוף |
| 1 | `.vp-hero` | תמונת מוצר + מחיר | קרוסלת צבעים, swatches, מחיר/הנחה, trust |
| 2 | `.s2` | מבצע | טיימר LIVE, סטטיסטיקות, פס התקדמות, תמונת וידאו |
| 3 | `.s3` | יתרונות + מפרט | גריד 2×2 יתרונות, טבלת מפרט טכני |
| 4 | `.s4` | ביקורות | דירוג 4.8, 3 כרטיסי ביקורת מאומתת |
| 5 | `.s5` | שאלות נפוצות | אקורדיון (`<details>`) — 5 שאלות |
| 6 | `.s8` | המכולה בדרך | CTA לגלריית מפעל + trust |
| 7 | `.s6` | גלריית תמונות | גריד תמונות + lightbox |
| 8 | `.s7` | CTA סגירה | בלוק navy כהה, מחיר, כפתור וואטסאפ זוהר |
| — | `.vp-footer` | פוטר | פרטי קשר, שעות, מפת SVG, רשתות, badges |
| — | `.vp-sticky` | כפתור צף | מחיר + "הזמינו עכשיו" — תמיד גלוי |

> הערה: סדר ה-DOM אינו תואם את מספור ה-class (`s8` לפני `s6`/`s7`) — המספור הוא היסטורי, הסדר הוויזואלי הוא הקובע.

## 4. קומפוננטות מרכזיות

### 4.1 Header (`.vp-header`)
- `position: sticky`, רקע שקוף-לבן `rgba(255,255,255,0.94)` + `backdrop-filter: blur(16px)`.
- גריד 3 עמודות: `44px | 1fr | 44px` (וואטסאפ | לוגו | שיתוף).
- לוגו "VIPO massage" עם נקודת gold כסיומת (`::after`).

### 4.2 Hero (`.vp-hero`)
- **Stage** (`aspect-ratio: 4/3`): תמונת מוצר עם מעבר fade+scale בהחלפה (`.vp-out`).
- **בקרת צבעים:** dots (מתרחבים כשפעילים) + swatches עגולים (cream/gray/black).
- **תמחור:** מחיר עכשיו `₪2,900`, מחיר קודם מחוק `₪12,000`, badge "76% הנחה" בזהב.
- **Trust strip:** 3 תאים (אחריות / CE / משלוח) בתוך גבול אחד מחולק.

### 4.3 כרטיס מבצע (`.s2-card`)
- Glassmorphism: `rgba(255,255,255,0.9)` + `blur(6px)`, צל רך, hover מרים.
- **טיימר:** 3 בלוקים navy (שעות/דקות/שניות) עם `tabular-nums`, JS סופר לאחור.
- **סטטיסטיקות:** רכישות / במכולה / נותרו — התא "נותרו" מודגש בזהב (`.s2-stat-alert`).
- **פס התקדמות:** מילוי gradient navy→`#3a7bd5` עם אנימציית מילוי בכניסה.
- **תמונת וידאו:** `aspect-ratio: 16/9`, overlay כהה, כפתור play, תווית משך.

### 4.4 כרטיס סקשן משותף (`.sec-card`)
בסיס חוזר לסקשנים 3/4/5/7: glass לבן `0.85` + `blur(8px)`, radius `18px`, hover מרים.
- כותרת (`.sec-card-title`) עם קו תחתון gradient navy→gold (`::after`).

### 4.5 יתרונות (`.s3-grid` / `.s3-ben`)
- גריד 2×2. כל כרטיס: אייקון צבעוני סמנטי + כותרת + תיאור.
- hover: רקע לבן + הרמה + צל.

### 4.6 מפרט טכני (`.s3-specs`)
- טבלה עם שורות מפוספסות (`nth-child(even)` רקע gold-רך), עמודה ימנית navy מודגשת.

### 4.7 ביקורות (`.s4-card`)
- אווטר עגול עם אות, שם, מיני-כוכבים, badge ירוק "רכישה מאומתת".
- גוף בנטוי עם גרשיים זהב דקורטיביים (`::before` = `"`).

### 4.8 שאלות נפוצות (`.s5-item`)
- `<details>/<summary>` נייטיב. פתוח → רקע לבן + פס gold ימני + חץ מסתובב (Font Awesome `\f078`).

### 4.9 CTA סגירה (`.s7-inner`)
- בלוק navy כהה (gradient), orb זהב דקורטיבי (`::before`).
- כפתור לבן עם אנימציית זוהר (`s7glow`) שמושכת תשומת לב.

### 4.10 גלריה + Lightbox (`.s6`)
- גריד: תמונה גדולה (`16/10`, רוחב מלא) + 2 ריבועיות (`1/1`).
- Lightbox: overlay כהה `blur(10px)`, תמונה ממורכזת, כפתור סגירה. נפתח/נסגר ב-JS.

### 4.11 Footer (`.vp-footer`)
- רקע navy כהה, פינות עליונות מעוגלות `24px`, פס gold עליון.
- לוגו + tagline, 2 עמודות (צרו קשר / שעות), **מפת SVG מיניאטורית** (כבישים, בלוקים, pin זהב + pulse), כפתורי רשתות, בר תחתון (badges + זכויות + לינקים משפטיים).

### 4.12 כפתור צף (`.vp-sticky`)
- `position: fixed` בתחתית, רוחב מקסימלי `480px`, `blur(12px)`.
- מחיר + כפתור וואטסאפ. תומך `env(safe-area-inset-bottom)` (iPhone notch).

## 5. אנימציות ואינטראקציות

| שם | טריגר | אפקט |
|----|-------|------|
| `.reveal` → `.is-visible` | IntersectionObserver (threshold 0.15) | fade + slide-up `24px`, `cubic-bezier(0.22,1,0.36,1)` |
| `reveal-delay-1/2/3` | גריד יתרונות | stagger 0.1/0.2/0.3s |
| `softPulse` | orbs ברקע (`main::before/after`) | נשימה איטית 8–10s |
| `s2p` | נקודת LIVE | פעימת halo 1.5s |
| `s2fill` | פס התקדמות | מילוי מ-0 בכניסה |
| `s7glow` | כפתור CTA סגירה | זוהר זהב מחזורי 2.5s |
| `ftPulse` | pin במפת הפוטר | התרחבות + דעיכה 2s |
| קרוסלת hero | אוטומטי 2s, נעילה 120s בלחיצה | fade+scale בהחלפת תמונה |

**מצב גלילה:** `scroll-behavior: smooth` גלובלי.

## 7. נגישות (Accessibility)

נקודות קיימות:
- `aria-label` על כפתורי אייקון (וואטסאפ, שיתוף, רשתות).
- HTML סמנטי: `<header>`, `<main>`, `<section>`, `<footer>`, `<details>`.
- `alt` על כל התמונות.

לבדיקה / שיפור עתידי:
- ניגודיות טקסט מעומעם (`--ink-3` על לבן) — לבדוק מול WCAG AA.
- מצב `prefers-reduced-motion` — כרגע לא ממומש; כדאי לעקוף אנימציות reveal/pulse.
- ניווט מקלדת ל-lightbox (Esc לסגירה).
