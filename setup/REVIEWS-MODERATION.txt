ביקורות — קישור אישי + אישור (אופציה ב)
============================================

איך זה עובד
-----------
1. לקוח מזמין → נוצר טוקן + קישור אישי
2. הקישור מופיע בהודעת WhatsApp (העבר ללקוח אחרי אישור)
3. הלקוח כותב ביקורת → נכנסת לגיליון ReviewsPending כ"ממתין"
4. אתה מאשר → משנה סטטוס ל"מאושר" → מופיע באתר
5. ביקורת רעה / ספאם → "נדחה" (לא באתר)

קישור לדוגמה:
https://vipogroup.github.io/3Massage-for-sale-VC/review.html?token=XXXXXXXX

התקנה (פעם אחת)
-----------------
1. Google Apps Script → הדבק setup/google-apps-script-stock-order.gs
2. SHEET_ID + REVIEW_SITE_URL — עדכן אם צריך
3. Deploy → Web app → New version → Anyone

אישור ביקורות בגיליון ReviewsPending
---------------------------------------
| status     | מה קורה        |
|------------|----------------|
| ממתין      | לא באתר        |
| מאושר      | מופיע באתר     |
| נדחה       | לא באתר        |

גיליונות חדשים:
- ReviewTokens — טוקנים לקישורים
- ReviewsPending — ביקורות + סטטוס

שליחת קישור ידנית
------------------
מגיליון ReviewTokens — העתק token ובנה:
review.html?token=TOKEN

או מעמודה reviewToken ב-Orders
